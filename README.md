# Vermanga Extension Repo

Repositorio de extensiones personalizadas para [Mihon](https://mihon.app).

## Instalar en Mihon

1. Abrí Mihon
2. Andá a **Browse → Extensiones**
3. Tocá el ícono de ajustes (tuerca) arriba a la derecha
4. Tocá **Editar repos**
5. Agregá esta URL:

```
https://raw.githubusercontent.com/TU_USUARIO/vermanga-repo/main/index.min.json
```

> ⚠️ Reemplazá `TU_USUARIO` con tu usuario de GitHub

6. Guardá y volvé a **Browse → Extensiones**
7. Buscá **Vermanga** e instalá
